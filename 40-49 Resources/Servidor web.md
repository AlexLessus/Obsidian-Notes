### Aplicación web
Una aplicación web genera páginas web interactivas que contienen varios tipos de lenguajes de marcas (HTML, XML) 

#### Arquitectura de dos niveles
Es la mas simple, se tiene el nivel del 'Cliente' y el nivel del servidor

#### Arquitectura de tres niveles
1. Capa de presentación, da a los datos un formato adecuado
2. Referido a algún tipo de programa o script
3. Proporciona al segundo los datos necesarios para su ejecución
