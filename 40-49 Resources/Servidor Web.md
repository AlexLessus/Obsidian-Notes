## Servidor proxy
Un **servidor proxy** es un sistema o aplicación especializada que actúa como **intermediario** entre un usuario (cliente) y los recursos de Internet, filtrando y reenviando el tráfico para mejorar la seguridad, la privacidad y el rendimiento.  

Su función principal es **ocultar la dirección IP real** del usuario, permitiendo el anonimato, eludir restricciones geográficas y proteger la red interna de amenazas externas.

## Tipos de servidores
- Servidor HTTP Apache
- Servidor Nginx
	- Igual que apache, funciona mejor que apache en la version de paga
- LiteSpeed
	- Sustituto de apache, para mejorar el rendimiento
- Microsoft IIS
	- Natico de windows, permite trabajar maquina personal para host
	- Solo para aplicaciones pequeñas
- Sun Java System Web Server
	- Para aplicaciones de java, python o Ruby
	- Soporta gran carga de trabajo con tecnologías especificas de java
