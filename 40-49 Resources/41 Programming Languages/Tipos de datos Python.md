### Estructurados (secuencias)
#### Secuencias de texto
	Cadenas(str) 'hola', "hola" str()
	str(38) # lo convierte a cadena
#### Secuencias
	Tuplas(tuple)(1,2,3,4)
	Listas(list)[1,2,3,4]

#### Secuencia numérica
	Rangos(Range) range(5) genera la secuencia: 0,1,2,3,4
#### Secuencia binaria
	Cadena binaria (bytes): b'hola'
	Cadena binaria multiple: (bytearray)
	Cadena 
### Mapas
	Diccionarios(dict)
