# Note Taking


- **Notas atómicas**: Cada nota debe contener una única idea o concepto (por ejemplo, un algoritmo específico como "Algoritmo de Dijkstra" o una función).
- **Identificadores únicos**: Cada nota tiene un ID único para una vinculación clara.
- **Enlaces extensivos**: Se conectan notas relacionadas (por ejemplo, "Capas Convolucionales" se puede vincular a "Clasificación de Imágenes" y "Funciones de Activación").
- **Escribir con tus propias palabras**: Esto es crucial para la comprensión en campos complejos como el aprendizaje automático, ya que evita la copia pasiva.
- **Etiquetas y estructura**: Se usan etiquetas (ej. #python, #pytorch, #computerVision) y estructuras de carpetas amplias, pero la conexión profunda de ideas se logra a través de los enlaces entre notas.
